<h3> Features </h3>
<ul>
  <li>Multi-Client Support</li>
  <li>Username Selection</li>
  <li>MOTD on Connect</li>
  <li>Connected Userlist</li>
  <li>Basic Animation</li>
  <li>Able to Set Status (Away / Busy / Online)</li>
</ul>
